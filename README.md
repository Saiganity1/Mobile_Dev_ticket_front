# Ticketing Frontend (Expo)

This is a minimal Expo React Native app that posts tickets to the Django backend and allows simple chat via polling.

Run

- install dependencies:

  npm install

- start Expo:

  npm start

Important

- The app uses API_BASE = http://10.0.2.2:8000/api which works for Android emulator pointing to host machine. Change to your backend host when testing on device or iOS.
